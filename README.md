# ソロワーズ - 無料パズルゲーム

## 概要

「ソロワーズ」は、ブラウザで無料で遊べるダイヤモンド型パズルゲームです。

同じ数字を隣接させてクリアを目指すシンプルながら奥深いゲームです。インストール不要で、スマートフォンでもPCでもすぐに遊べます。

## 🎮 ゲームの特徴

### シンプルなルール、奥深い戦略
* **誰でもすぐに理解できる**：同じ数字を隣接させるだけのシンプルなルール
* **奥深いパズル性**：簡単に見えて、実は考えさせられる戦略的なゲーム
* **何度も挑戦したくなる**：試行錯誤を繰り返すことで、最適な解法が見えてくる

### 多彩なゲームモード
* **10段階のレベル**：初心者から上級者まで楽しめる難易度設定
* **クリエイトモード**：自分だけのオリジナルマップを作成可能
* **オンラインランキング**：タイムランキングと手数ランキングで全国のプレイヤーと競争

### プレイヤーに優しい設計
* **年齢層に応じた設定**：シニアやお子様向けの広告なしモード搭載
* **タイマーON/OFF機能**：自分のペースでじっくり考えたい方に最適
* **色付き駒機能**：同じ数字を色分けして見やすく表示

### アナログでも遊べる
* **紙とペンで完全再現**：デジタル版だけでなく、紙に書いて遊ぶこともできます
* **どこでも楽しめる**：外出先でも、電源がなくても楽しめる設計

## 📖 基本ルール

1. **目的**：同じ数字の駒をすべて隣接させる
2. **移動**：駒は斜め方向に移動可能
3. **固定**：同じ数字が隣接すると、その駒は光って固定される
4. **クリア**：すべての駒を固定できればクリア！

## 🎯 こんな人におすすめ

* **パズルゲームが好きな方**：論理的思考力を鍛えながら楽しめます
* **スキマ時間に遊びたい方**：1ゲーム数分で気軽にプレイ可能
* **競争が好きな方**：オンラインランキングで全国のプレイヤーと競えます
* **クリエイティブな方**：自分だけのマップを作って楽しめます
* **家族で遊びたい方**：紙とペンでアナログでも楽しめます

## 🚀 プレイ方法

インストール不要、URLをクリックするだけですぐに遊べます。

[Webゲームはこちら](https://solowords-yuki.github.io/)

## 💡 攻略のコツ

### 初心者向け
* まずは全体を見渡して、どこにどの数字があるか確認
* 近くにあるペアから順番に揃える
* 詰まっても大丈夫！何度でもリトライ可能

### 中級者向け
* 2〜3手先を読む習慣をつける
* 固定する順番を慎重に考える
* 道を塞がないように注意

### 上級者向け
* クリア状態から逆算して手順を組み立てる
* 複数の解法を比較して最適解を探す
* 操作スピードを上げてタイムランキング上位を狙う

詳しくは[攻略のコツページ](https://solowords-yuki.github.io/tips.html)をご覧ください。
